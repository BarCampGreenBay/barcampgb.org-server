BarCamp Green Bay Server
========================

BarCamp Green Bay server provisioning code.

## Installation

- Clone the source.
- `vagrant up`
- `vagrant ssh`
- `ifconfig` to grab the ip address
- load ip address in browser

## Planned Architecture

- Ansible for provisioning
- Nginx proxy
- Node process
- MongoDB process
